# Ham's Student Thesis Code Hub

Welcome to our cozy corner of code! 🎉  
This repo houses all the awesome thesis implementations from [Ilham G. Adillion](https://scholar.google.co.id/citations?user=KJHvkTUAAAAJ&hl=en)'s students at [Informatics Department of Institut Teknologi Sepuluh Nopember](https://www.its.ac.id/informatika/) (ITS - Surabaya, Indonesia).

We’re all about **open learning, collaboration, and reproducibility**—so feel welcome to explore, tinker, and build on these projects! Just remember to give credit where it’s due if you find something helpful.

Happy coding, and enjoy the journey! 🚀
